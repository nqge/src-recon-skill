### **示例 1: 高价值主机**

```
🔥 高价值主机（全端口开放）
─────────────────────────────
主机: 123.103.115.72
开放端口: 9 个
  ├─ 80/tcp    (HTTP)       - Web 服务
  ├─ 443/tcp   (HTTPS)      - Web 服务
  ├─ 3000/tcp  (Node.js)    - 开发端口 ⚠️
  ├─ 5000/tcp  (Flask)      - 开发端口 ⚠️
  ├─ 8888/tcp  (HTTP Alt)   - 开发端口 ⚠️
  └─ 9000/tcp  (HTTP Alt)   - 开发端口 ⚠️

风险等级: 🔴 高
原因:
  - 全端口开放
  - 开发端口暴露
  - 可能存在开发接口泄露
  - 可能存在未授权的管理后台

建议:
  - 禁止公网访问开发端口
  - 配置防火墙规则
  - 检查管理后台配置
```
